To add and commit in one line->git commit -a -m "commit message"
To show the braches-> git branch
To crate new branch-> git branch branch-name
Switch to new branch-> git checkout branch-name
To create new branch and switch to it ->git checkout -b branch-name
To delete a branch-> git branch -d branch-name
To forcibly delete a branch -> git branch -D branch-name
To merge to branch-> git merge branch-name 
To abort merge-> git merge --abort
To view summarize history of comit -> git log --graph --oneline
To view branch diagram -> git log --all --decorate --online --graph
To create new branch in remote repository -> git push -u origin branch-name
To remove branch in remote repo => git push --delete origin branch-name
To rebase a branch -> git rebase branch-name
To rebase mulitple commits -> git rebase -i master (squaf)
to force git to push => git push -f
to pull repository => git pull
